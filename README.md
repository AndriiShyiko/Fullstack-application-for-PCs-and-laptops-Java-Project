Development: Completed.
Description: A full working application for Laptops or PC built on Swing Java and connected to the DB.
Instruction on how to run Application: Create a DB as in the DB dump, Download "FullstackJava - copy" fully following the files and directory placements, replace 2 fields in the CommonConstants.java with your own username and password for the DB, click 2 times on "FullstackJavaJAR_EXE.bat" to run Application. You may need to remake jar file because of the username and password replacement.
DB used for this project is MySQL Workbench.
